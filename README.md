  - Provides user authentication with Firebase.
  - Supports new account creation using email and password.
  - Validates password length (6+ characters) during sign-up.
  - Allows registered users to sign in.
  - Displays the user's email on the home screen after login.
  - Includes a logout button to sign the user out.
  - Navigates back to the login screen after logout.
  - Built with a modern UI using Jetpack Compose.
  - Features a "Show/Hide Password" toggle for usability.
  - Shows clear login and registration error messages directly on the screen.
